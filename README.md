# LendInvest

LendInvest Challenge

### Installing

```sh
$ git clone https://gitlab.com/seechrobin/lendinvest-coding-challenge
$ cd lendinvest-coding-challenges
$ yarn install
```

## Running

```sh
$ yarn start
```

## Testing

```sh
$ yarn test
```
